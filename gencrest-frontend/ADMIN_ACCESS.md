# Admin Portal Access

## Admin Login

Access the admin portal at: **`/admin`**

The admin portal provides full system access with elevated privileges.

## Admin Credentials

### Primary Admin Account
- **Username**: `admin`
- **Password**: `admin`
- **Role**: ADMIN
- **Access Level**: Full system access

### Secondary Admin Account (SFA Admin)
- **Username**: `sfaadmin`
- **Password**: `sfaadmin`
- **Role**: sfaadmin
- **Access Level**: Full system access

## Features

The admin portal login provides:
- Dark-themed secure interface
- Full access to all system modules
- Access to Technical Documentation
- User Management capabilities
- System configuration options
- All reports and analytics

## Security Notes

- All admin activities are monitored and logged
- Admin access should be restricted to authorized personnel only
- The admin portal uses the same authentication backend
- Consider changing default credentials in production environments

## Accessing the Portal

1. Navigate to `/admin` in your browser
2. Enter admin credentials
3. Click "Access Admin Portal"
4. You'll be redirected to the dashboard with full access

Alternatively, from the regular login page, click the "Admin Portal" link at the bottom.
